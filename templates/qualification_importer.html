Spreadsheet Rules
  • Column B: NEMO tool ID.
  • Column C: qualification date. Time is ignored.
  • Column D: user email address.
  • User email is matched against /api/users/.
  • Each qualification is posted to /api/qualifications/ with qualification_level = 1.

The uploaded file is processed temporarily for this request and then deleted. The token is used only to call the NEMO API for the current run.
